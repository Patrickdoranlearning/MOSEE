# Plans

Implementation plans produced by `jimmy plan` / `jimmy dual-plan` / `jimmy build`.

- `PLAN-[name].md` — full plans (complex features, schema, methodology). Header carries `Status: Draft → Ready` (Karen-approved) and `Revision: N`.
- `MINI-[name].md` — mini plans (simple features, complex bugs).
- `completed/` — archived after shipping (moved by wrap-up Pillar 4 via `git mv`).

A plan is executable only at `Status: Ready`. See `.claude/agents/planner.md` for the format and `.claude/agents/jimmy.md` for the Karen review protocol.
